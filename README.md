## 1️⃣ 스터디 소개
<br/>

- 알고리즘 실력 향상을 위해 만든 Beakjoon문제풀이 스터디 Level-UP입니다!
- 주 2회 주어진 Beakjoon문제를 풀고 자신의 풀이 방법을 소개하고 팀원들의 풀이를 리뷰합니다.
- 스터디 이름인 Level-UP에 맞추어서 난이도가 낮은 문제부터 도전하여 점차 높은 난이도의 문제를 도전합니다.
- JavaScript로 문제를 풀어요!
